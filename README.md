## Vuetify starter project for [Code App](https://thebaselab.com/code)

To start, run these commands in the terminal:

```bash
npm i       # Install dependencies.
npm run dev # Start the development server.
```

To preview the project, open Safari in split view and visit `http://localhost:3000`.

Support commands:

```bash
npm run dev    # Start the development server.
npm run build  # Build source code.
npm run format # Format source code.
```
